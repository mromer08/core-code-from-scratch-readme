# Exercise
Write a function `greet` having one parameter and returning `'Hello <parameter>!'`.

Example: `greet('Ada')` should return `'Hello Ada!'` and `greet('Grace')` should return `'Hello Grace!'`.

# Solution

```js
function greet (param) {
    return 'Hello ' + param + '!';
}
```
