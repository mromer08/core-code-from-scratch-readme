# Exercise
1. Define a function `greet` returning the value `'Haydo!'`.
2. Declare a variable `salutation`. Call the function `greet` and assign the result of the call to the variable `salutation`.

# Solution
```js
function greet () {
    return 'Haydo!';
}

let salutation = greet();
```
