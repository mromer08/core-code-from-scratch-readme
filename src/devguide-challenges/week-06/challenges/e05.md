# Exercise

Which value does `x` have after execution of the following code?

```js
let x = 'Laurel';
let y = 'Hardy';
let z = y;
y = x;
x = z;
```

# Solution
```js
'Hardy'
```
> Because z gets the value 'Hardy' from y and finally x gets the value from z
